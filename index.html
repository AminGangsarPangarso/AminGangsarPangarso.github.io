My Projects

Hypertext Markup Language atau HTML adalah bahasa markup standar yang digunakan untuk membuat halaman website dan aplikasi web.

CSS dipakai untuk mendesain halaman depan atau tampilan website (front end). CSS menangani tampilan dan ‘rasa’ dari halaman website.

Web Design adalah suatu kegiatan yang dilakukan untuk membuat sekaligus merancang desain web yang berkaitan dengan bagaimana konten akan ditampilkan kepada pengunjung website.

Layout adalah desain tata letak, sedangkan arti dari layout adalah suatu susunan, rancangan, atau tata letak rang dari sebuah elemen yang sengaja didesain untuk bisa ditempatkan dalam suatu bidang yang sebelumnya telah direncanakan sistemnya terlebih dahulu.

WordPress bisa membantu Anda membuat dan mengelola website tanpa koding. Anda bisa menata tampilan, menambahkan fitur, dan membuat konten di website dengan mudah.
